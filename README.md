# asset-manager（統合資産管理システム）

タグ・マスタ定義・カスタムフィールドに対応した、ブラウザ完結型の資産管理システムです。
データはブラウザの IndexedDB（Dexie.js）に自動保存され、サーバーは不要です。

## 起動方法

ビルドは不要です。いずれかの方法で開いてください。

```bash
# 方法1: そのままブラウザで開く
open index.html

# 方法2: ローカルサーバーで配信（推奨）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 主な機能

- **ダッシュボード** — 稼働率などの統計、期限アラート、最近の資産・履歴
- **資産一覧** — リスト / カード / カレンダーの3ビュー、列のカスタマイズ（表示設定）、列ソート
- **資産詳細** — 基本情報、画像（ドラッグ&ドロップ対応）、構成パーツ（親子関係）、運用手順、リスク管理、周期計画、履歴
- **マスタ管理** — タグの階層管理（リスト / グラフの2ビュー）、タグ固有のカスタムフィールド定義
- **資産構成** — 資産同士の親子関係をグラフで編集
- **検索・フィルタ** — 資産名 / タグ / ID の横断検索（`Ctrl+K`）、サイドバーからのタグ絞り込み
- **インポート / エクスポート** — JSON形式（画面へのドラッグ&ドロップでもインポート可能）

### キーボードショートカット

| キー | 動作 |
| --- | --- |
| `Ctrl (⌘) + K` | 検索ボックスへフォーカス |
| `Esc` | 最前面のモーダル / モバイルサイドバーを閉じる |
| `Delete` | グラフビューで選択中のノード・接続を削除 |

## プロジェクト構成

かつては単一の `index.html`（約9,000行）でしたが、機能ごとに CSS / JavaScript を分割しています。

```
asset-manager/
├── index.html              # HTML骨格（マークアップのみ）
├── favicon.svg
├── css/
│   ├── base.css            # デザイントークン（CSS変数）・共通部品（ボタン/バッジ/入力欄）
│   ├── layout.css          # サイドバー・ヘッダー・メイン領域のレイアウト
│   ├── components.css      # 資産カード/リスト/検索/ダッシュボード/カレンダー/トースト
│   ├── modal.css           # モーダル・タブ・タグ入力（オートコンプリート）
│   ├── master-editor.css   # マスタ管理（リストエディタ/グラフビュー/ノード）
│   └── responsive.css      # レスポンシブ対応（モバイルサイドバー等）
├── js/                     # 読み込み順 = 依存順（index.html の下部を参照）
│   ├── config.js           # 初期データ・ステータス定義(STATUS_DEFS)・DBスキーマ・永続化
│   ├── utils.js            # 汎用ユーティリティ（タグ階層/日付/トースト/バッジ生成 等）
│   ├── ui.js               # モーダル開閉・アコーディオン・サイドバー開閉
│   ├── navigation.js       # サイドバー描画・ページ遷移
│   ├── search.js           # 検索・タグ/マスタによるフィルタ
│   ├── dashboard.js        # ダッシュボード描画
│   ├── maintenance.js      # 履歴・周期計画のCRUD
│   ├── import-export.js    # JSON入出力・画像管理
│   ├── assets.js           # 資産一覧（リスト/カード/カレンダービュー）
│   ├── asset-modal.js      # 資産詳細モーダル・カスタムフィールド入力
│   ├── master.js           # マスタ管理モーダル（タグ階層のリストエディタ）
│   ├── graph.js            # グラフビュー（タグ/資産構成のノードエディタ）
│   ├── list-settings.js    # リスト表示の列設定
│   └── main.js             # 初期化・グローバルイベント（ショートカット等）
└── vendor/
    ├── dexie.min.js        # Dexie.js v3.2.4（IndexedDBラッパー、オフライン動作のため同梱）
    └── DEXIE-LICENSE
```

## カスタマイズのポイント

- **ステータスの追加・変更** — `js/config.js` の `STATUS_DEFS` に1行追加するだけで、
  資産編集画面のチェックボックス・一覧のバッジ色・資産構成グラフのノード色に自動反映されます。
- **配色・角丸などのデザイン** — `css/base.css` の `:root` にあるCSS変数を変更してください。
- **リストの列** — 画面上の「表示設定」から変更できます（定義は `js/main.js` の `availableColumns`）。

## 依存関係

| ライブラリ | 提供元 | 用途 |
| --- | --- | --- |
| Dexie.js 3.2.4 | 同梱（`vendor/`） | IndexedDBによるデータ永続化 |
| Font Awesome 6.4.0 | CDN | アイコン |
| Google Fonts (Inter / Noto Sans JP) | CDN | フォント |

データの保存・読み込みはオフラインでも動作します（アイコン・フォントのみCDN依存のため、
完全オフラインで使う場合は Font Awesome も同様に `vendor/` へ同梱してください）。
